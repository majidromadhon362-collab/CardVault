# User Guide

## Workflow Harian

1. Selesai dokumentasi/event, masukkan MMC/SD card ke laptop.
2. Buka CardVault.
3. Saat pertama kali buka, baca onboarding step-by-step lalu klik `Mulai Pakai App`.
4. Klik `Tambah File`.
5. Pilih footage yang ingin dibackup.
6. Pilih preset `Balanced` untuk penggunaan normal.
7. Klik `Pilih Folder Output` dan pilih folder project di laptop.
8. Klik `Mulai Kompres`.
9. Tunggu sampai selesai.
10. Putar beberapa hasil output untuk validasi.
11. Baru pindahkan/hapus original secara manual kalau sudah yakin aman.

## Pilihan Preset

### High Quality

Untuk footage penting, video client, wedding, event besar, atau file yang mungkin akan diedit lagi.

Kelebihan:

- Kualitas visual paling aman.
- Cocok untuk archive.

Kekurangan:

- Ukuran hasil bisa lebih besar dari preset lain.
- Proses lebih lambat.

### Balanced

Rekomendasi default untuk kebanyakan footage.

Kelebihan:

- Ukuran turun cukup signifikan.
- Kualitas masih aman untuk dilihat dan diarsipkan.
- Tidak seagresif preset small.

### Small Size

Untuk footage dokumentasi yang tidak terlalu kritikal atau footage panjang yang hanya perlu disimpan sebagai backup referensi.

Kelebihan:

- Ukuran paling hemat.

Kekurangan:

- Risiko detail visual lebih turun.

## Tips Aman

- Jangan langsung format MMC setelah kompres.
- Simpan original sampai hasil sudah dicek.
- Untuk project penting, simpan backup kedua di external drive/cloud.
- Cek durasi output sama dengan source.
- Cek audio tetap ada.

## Kenapa Hasil Bisa Tidak Selalu Lebih Kecil?

Kalau source video sudah sangat terkompres, encode ulang bisa saja tidak menghemat banyak ukuran. Ini normal. Preset bisa dibuat lebih agresif, tapi risikonya kualitas turun.
